Buat Short Link

Buat tautan pendek menggunakan GitHub Issues API

Mekanisme:

  • Title Issue = URL tujuan (harus valid URL)
  • Body Issue = Judul/deskripsi link
  • Issue Number = Path parameter untuk short link
URL harus valid dan lengkap dengan http/https

Sedang membuat GitHub Issue...

Berhasil!

Info GitHub Token:

Buat Personal Access Token di: GitHub Settings → Developer settings → Personal access tokens

Permission yang dibutuhkan: repo (untuk membuat issue)

Catatan Penting:

  • Pastikan repository GitHub Anda sudah ada
  • Token harus memiliki permission "repo"
  • URL tujuan harus valid (dimulai dengan http:// atau https://)
  • Issue number akan menjadi path parameter short link
