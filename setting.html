Buat Short Link

Buat tautan pendek menggunakan GitHub Issues API

Cara Menggunakan:

  1. Buat Personal Access Token di GitHub dengan permission "repo"
  2. Isi form di bawah dengan URL tujuan dan token Anda
  3. Klik "Buat Short Link" untuk membuat issue baru
  4. Issue number akan menjadi path parameter untuk short link

Sedang membuat short link...

Berhasil!
